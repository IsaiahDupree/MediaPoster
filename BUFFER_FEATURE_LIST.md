## üì§ PUBLISH Module

### Core Scheduling Features
- **Multi-Platform Scheduling:** Schedule to all 11 platforms from one dashboard
- **Visual Content Calendar:** Week/month view of scheduled content
- **Queue Management:** Create, re-arrange, or shuffle post queues
- **Best Time to Post:** AI-powered posting time recommendations
- **Reminder Notifications:** Mobile alerts for platforms requiring native posting

### Post Types Supported
- **Threaded Posts:** X, Bluesky, Threads, Mastodon
- **Carousels:** Instagram, LinkedIn
- **Videos:** All platforms
- **Stories:** Instagram (via reminder)
- **Reels:** Instagram (via reminder or direct)

### Advanced Publishing Features
- **First Comment Scheduling:** Instagram & LinkedIn (Essentials+)
- **Custom Video Covers:** Facebook, Instagram, TikTok
- **Link Shortening:** Automatic URL shortening
- **Duplicate Posts:** Clone and reuse posts
- **Channel Groups:** Group channels for bulk selection
- **Hashtag Manager:** Save and reuse hashtag sets
